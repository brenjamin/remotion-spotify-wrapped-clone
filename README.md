# Remotion Spotify Wrapped video

- A Spotify Wrapped video made with Remotion
- Based on [this Youtube Tutorial](https://youtu.be/I-y_5H9-3gk)
- The final mp4 video can be found at out/video.mp4

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm start
```

**Render video**

```console
npm run build
```

**Upgrade Remotion**

```console
npm run upgrade
```
